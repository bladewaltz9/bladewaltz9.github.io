2 Github Page 配置

前期准备：

  1. 安装 git
  2. 注册 github 账号
  3. 配置 github ssh 密钥
  1. 新建一个仓库，仓库名称是你的 github 用户名 + .github.io 后缀，即 yourname.github.io。README 初始化也可以选上，会自动生成一个分支。
  2. 进入 Setting - Pages，就可以看到自己的 github page 的网址，点进去就可以看到一个初始化页面。

4 Hexo 使用

4.1 写文章和发布文章

  1. 新建一篇文章：hexo new post "title"，然后在 ./source/_posts 目录下，会生成一个 title.md 文件，然后可以编写文章内容。

  2. 使用 hexo g 或 hexo generate 生成静态网页。

  3. 使用 hexo s 或 hexo server 可以在本地查看效果。

  4. 使用 hexo d 或 hexo deploy 上传到 github，然后主页就可以看到发布的文章。

4.2 图片不显示（图片路径问题）

hexo文章中添加本地图片不显示

图片路径主要分为绝对路径、相对路径和网络路径。本文采用相对路径的方式：

  1. 修改 _config.yml 里的 post_asset_folder 选项为 true。

    修改后，创建新的文件时会在文章目录下生成一个同名文件夹，用来存储图片。

  2. 修改 typora 设置，偏好设置 > 图像 > 修改图片路径，之后插入图片时会复制到指定路径中。

  3. 安装 hexo-asset-image 插件：

    网上有人说该插件有遇到问题，但我使用没有问题，应该是版本原因。

  4. md 中的格式采用相对路径：![test](testimages/test.jpg)

  5. 注意如果重命名 md 文件，记得要重命名图片文件夹。
